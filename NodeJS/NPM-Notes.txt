// # Node PACKAGE MANAGER

----> Чтобы установить пакет  в папке через Гит-Баш, но при этом не записывает в PACKAGE.JSON
npm install namePackage

----> Чтобы установить пакет  в папке через Гит-Баш, а так записывает в PACKAGE.JSON
----> (зачем? для того чтобы следить и знать какие пакеты установены)
npm install namePackage -save


----> Чтобы удалить пакет  в папке через Гит-Баш
npm uninstall namePackage
